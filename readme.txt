INTRODUKTION:
    Der findes 4 standard sider.
        1. Den statiske forside med seneste oplæg og produkter på.
        2. Om siden.
        3. Arkiv af alle posts, varer, indlæg, artikler, mm.
        4. Single listing, i.e., et produkt.


    Kategori sider, i.e., Møbler, viser alle underkatogorier e.g., Teak, Stole, Birk, etc.


################################################################################################################################################################
NYT PRODUKT:
    Når der ligges nye produkter ind skal der vælges kategorier og sub-kategorier.
    Indsæt en header block - denne er prisen.
    Titlen på indlægget er varens navn.
    Excerpt er beskrivelsen af produktet.
    Billeder af produktet indsættes ved at ligge alle billeder ind i den rækkefølge man ønsker dem set som en image block hver for sig.
    Featured Image vælges manuelt.
